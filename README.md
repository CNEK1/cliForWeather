# cliForWeather

Basic CLI for checking weather.

Options: --version Show version number  
 -h, --help Show help  
 -s, --search Weather in search town  
 -t, --token Saving API token  
 -d, --daily Forecast for 5 days with data every 3 hours

Examples: weatherIndex.js -s <weather in search town>

THIS CLI uses FREE Open Weather API.
